# prspr-mobile-client

## Expo Publish

==> cd prspr

==> expo publish


## Expo Build
expo build:android 

or 

expo build:ios

## Steps

https://github.com/react-community/create-react-native-app

==> expo init prspr
==> cd prspr
==> npm start

==> npm run ios
Like npm start, but also attempts to open your app in the iOS Simulator if you're on a Mac and have it installed.

==> npm run android
